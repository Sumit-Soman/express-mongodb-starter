### Express-Mongodb-Starter

install

    $ npm install

run as application

    $ npm start

#### Features NPM modules

* Database: MongoDB (mongoose)

* Validation: Joi, Joi-objectid, 

* Restful API : Express

    POST : /api/user/register
    POST: /api/user/login
    GET: /api/user/home  

* Custom Middleware: handle for error, exception, log
